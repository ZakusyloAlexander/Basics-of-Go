# Лабораторна робота №4 — Виконані завдання

**Тема:** Робота з графічним інтерфейсом. Label, CheckBox, ComboBox, Entry, Button.

## Вимоги

- **Go** 1.21+
- **cgo** (компілятор C)
- **Windows:** mingw-w64 (http://mingw-w64.org)
- **Linux:** `sudo apt-get install libgtk-3-dev` (або gtk3-devel)
- **macOS:** Xcode Command Line Tools

Встановлення пакета:
```
go get github.com/andlabs/ui
```

## Структура

### 1. ex1-glass-go — Калькулятор склопакета (варіант 1, Go)

Розрахунок вартості склопакета функціями мовою Go.

- **Label** — ширина, висота, тип, матеріал
- **Entry** — ширина (см), висота (см)
- **ComboBox** — тип склопакета (однокамерний/двокамерний), матеріал рами
- **CheckBox** — підвіконня (+350 грн)
- **Button** — розрахувати

Ціни за 1 см²: однокамерний дерев'яний 2.5 грн, двокамерний дерев'яний 3 грн, однокамерний металевий 0.5 грн, двокамерний металевий 1 грн, однокамерний металопластиковий 1.5 грн, двокамерний металопластиковий 2 грн.

### 2. ex2-glass-cgo — Калькулятор склопакета (варіант 2, C + Go)

Те саме завдання, але обчислення в C (`calc.c`), Go-обгортка викликає C-функцію.

### 3. ex3-tour — Калькулятор туру

Розрахунок вартості туру на відпочинок.

- **ComboBox** — країна (Болгарія, Німеччина, Польща), сезон (літо, зима)
- **Entry** — кількість днів, кількість путівок
- **CheckBox** — індивідуальний гід (+$50/день), номер люкс (+20%)
- **Button** — розрахувати

Ціни за 1 день: Болгарія літо $100/зима $150, Німеччина літо $160/зима $200, Польща літо $120/зима $180.

## Як запускати

```bash
# Калькулятор склопакета (Go)
cd Lab4\Виконані завдання\ex1-glass-go
go run .

# Калькулятор склопакета (C + Go)
cd Lab4\Виконані завдання\ex2-glass-cgo
go run .

# Калькулятор туру
cd Lab4\Виконані завдання\ex3-tour
go run .
```

**Важливо:** Запускати з папки відповідного проєкту (де лежить go.mod).
