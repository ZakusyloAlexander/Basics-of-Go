# Лабораторна робота №2 — Виконані завдання

Кожен приклад має свій **go.mod** з унікальним іменем модуля. Запускати потрібно **обов'язково з папки відповідного прикладу**.

## Як запускати

### 1. ex1-onefile (пункт 3 — один файл)

Відкрийте термінал, перейдіть у папку **ex1-onefile** і виконайте:

```
cd Lab2\Виконані завдання\ex1-onefile
go run .
```

Або в IDE: правий клік по папці **ex1-onefile** → Open in Integrated Terminal → `go run .`

### 2. ex2-twofiles (пункт 4 — два файли)

```
cd Lab2\Виконані завдання\ex2-twofiles
go run .
```

### 3. ex3-package (пункт 5 — пакет calc)

```
cd Lab2\Виконані завдання\ex3-package
go run .
```

### 4. ex4-with-tests (пункт 6 — пакет і тести)

Запуск програми:

```
cd Lab2\Виконані завдання\ex4-with-tests
go run .
```

Запуск тестів:

```
cd Lab2\Виконані завдання\ex4-with-tests
go test ./calc/... -v
```

---

**Важливо:** Команди `go run` та `go test` мають виконуватися з папки, де лежить **go.mod** (ex1-onefile, ex2-twofiles, ex3-package або ex4-with-tests). Якщо запускати з кореня проекту або з іншої папки — буде помилка імпорту.

## Функції в усіх прикладах

- **Min3(a, b, c)** — мінімальне з трьох чисел
- **Average3(a, b, c)** — середнє арифметичне трьох чисел  
- **SolveLinear(a, b)** — розв'язок рівняння ax + b = 0 (x = -b/a)
