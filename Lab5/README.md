# Lab5 (Go)

Варіант: `непарний` (структури `Product` та `Currency`).

## Запуск
У папці `Lab5` виконайте:
`go run .`

## Формат вводу
1. `n` — кількість товарів.
2. Для кожного товару:
   - `Name` — назва товару
   - `Price` — ціна одиниці (у валюті `Cost`)
   - `Cost.Name` — назва валюти
   - `Cost.ExRate` — курс: скільки гривень за 1 одиницю валюти
   - `Quantity` — кількість
   - `Producer` — виробник
   - `Weight` — вага одиниці

Програма виведе список товарів, а потім найдешевший і найдорожчий (за загальною вартістю в гривнях).
