# Лабораторна робота №3 — Виконані завдання

**Тема:** Управляючі конструкції в мові Go. Функції. Введення та виведення. Генерування випадкових чисел.

**Варіант 1** (таблиця 3.2): a=1103515245, c=12345, m=2³¹, діапазон [0, 100), довжина послідовності K=10000.

## Що реалізовано

### Завдання 1
- Функція генерування цілочисельної послідовності псевдовипадкових значень (лінійний конгруентний метод: X_{k+1} = (a·X_k + c) mod m).
- Обробка масиву даних:
  1. Частота інтервалів (інтервал = 1).
  2. Статистична ймовірність P_i = L_i / K.
  3. Математичне сподівання M(X) = Σ x_i·p_i.
  4. Дисперсія D(X) = Σ p_i·(x_i − M(X))².
  5. Середньоквадратичне відхилення σ = √D(X).

### Завдання 2
- Функція генерування послідовності дійсних псевдовипадкових значень (той самий LCG, значення X/m масштабуються на діапазон [0, n)).
